# Personal Calendar
A very simple personal calendar that lets the user can able to plan things and keep on track of their schedule activities.


## Extra Features: 
1.) Preview 3D Earth 


## Require install packages:
```bash
1.) npm i
```

## Run Project:
you need your discord bot token
```bash 
1.) npm start 
```
